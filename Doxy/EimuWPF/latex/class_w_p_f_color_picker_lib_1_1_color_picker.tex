\subsection{Detailed Description}
A simple WPF color picker. The basic idea is to ise a Color swatch image and then pick out a single pixel and use that pixels RGB values along with the Alpha slider to form a SelectedColor. \hyperlink{class_w_p_f_color_picker_lib_1_1_color_picker}{ColorPicker}.

This class borrows an idea or two from the following source(s)

AlphaSlider, and Preview box Based on an article by ShawnVN's Blog \href{http://weblogs.asp.net/savanness/archive/2006/12/05/colorcomb-yet-another-color-picker-dialog-for-wpf.aspx}{\tt http://weblogs.asp.net/savanness/archive/2006/12/05/colorcomb-\/yet-\/another-\/color-\/picker-\/dialog-\/for-\/wpf.aspx}

1$\ast$1 pixel copy Based on an article by Lee Brimelow \href{http://thewpfblog.com/?p=62}{\tt http://thewpfblog.com/?p=62}
